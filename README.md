# 체스 미션

### 진영

- 진영은 흑팀과 백팀으로 나뉜다.
- 흑팀은 위쪽부터 배치한다.
- 백팀을 아래쪽부터 배치한다.

### 위치

- 위치는 가로와 세로의 좌표 값으로 결정된다.
- 가로는 왼쪽부터 a~h 이다.
- 세로는 아래부터 위로 1~8 이다.

### 기물

- 기물의 종류는 6가지이다. 
  - 룩(Rook)
  - 나이트(Knight)
  - 비숍(Bishop)
  - 퀸(Queen)
  - 킹(King)
  - 폰(Pawn)

### 입력

- 게임의 시작과 종료 여부를 입력받는다.

### 출력

- 체스 게임 시작 메세지를 출력한다.
- 게임을 시작하면 체스판을 출력한다.
- 흑팀은 대문자로, 백팀은 소문자로 표시한다.
- 비어있는 좌표는 . 으로 표시한다.
